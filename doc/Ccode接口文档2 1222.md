## 方案系统
### 获取方案Git信息
##### 发送参数:
Id - 方案Id
##### 返回参数:
code
msg
data:{
    description:'开源中国的Git',
    url:"http://oschina.net/jsmyw/a.git",
    cloning:true, -是否正在clone
    lastClone:"2016/6/9-12:00:00", -上次clone的时间
    info:'' - 成功信息,失败信息(正在clone，用户密码错误，无法获取git)
}

ide - IDE
time - 10(s为单位)

code
msg
